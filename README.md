# 1A2B

## Instructions

A stands for both the position and the number is correct.  
B stands for the number is correct, but position is not.

    For instance:  

    answer : 1234  
    guess  : 1567  
             ^  
             A  
    1A0B  

    answer : 1234  
    guess  : 1425  
             ^^^  
             ABB  
    1A2B
